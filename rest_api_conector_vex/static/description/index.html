Odoo Api Rest Connector

Enables a REST API for the Odoo server. With the API can generated token authenticate . Afterwards, a set of routes to interact with the server are provided. The API can be used by any language or framework which can make an HTTP requests and receive responses with JSON payloads and works with both the Community and the Enterprise Edition.
